# Lint
Lint - an Hybrid-Programing Language writed with Python which is based on C++, so the source code of .lint file will compile into C++ code.

# Examples:

### main.lint
```python
password: 120

loop code for 999 {
    if code == password {
        log('The password was: <code>')
        stop
    }
    log(code)
}
```
